Exercício 10: Pegando dados de uma Api

Quantos compartilhamentos um domínio tem?

Nesse exercício nós vamos usar o serviço do site https://www.sharedcount.com/ para descobrir quantos compartilhamentos um determinado domínio tem.

Para isso nós vamos usar a seguinte URL para fazer a requisição: 

      https://api.sharedcount.com/v1.0/?url={MINHA_URL}&apikey={API_KEY}

Onde MINHA_URL deve ser a url a ser pesquisada e API_KEY dever ser: 

      1934f519a63e142e0d3c893e59cc37fe0172e98a

Digite a url abaixo:

N° URLs Reações no Facebook Compartilhamento no Facebook
1 https://mesalva.com/ 5.500 2.500
2 https://mesalva.com/ 5.500 2.500
