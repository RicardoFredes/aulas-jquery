Exercício 01: DOM (Document Object Model)

O que é o DOM?

É uma árvore de elementos descritos em HTML, XML ou XHTML.

O que é JQuery

É uma biblioteca em Java Script que disponibiliza uma boa quantidade de métodos, que são fáceis de usar (aprendizado e implementação). O lema do JQuery traduz muito bem o que ele faz: WRITE LESS, DO MORE. (Escreva menos, faça mais). Essa biblioteca é executada no lado do cliente (Client Side), ou seja, quando o conteúdo da página já foi baixado para o dispositivo do Cliente e o navegador o renderiza.

Como acessar o DOM por JQuery

A biblioteca disponibiliza um método de seleção de elementos igual ao utilizado em CSS.

  • nomedatag: seleciona elementos pelo nome da tag.
  • #nomedoid: seleciona elementos pelo seu id.
  • .nomedaclasse: seleciona elementos pela sua classe.
